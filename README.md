# Fluxo Kawasaki & Despachante
Projeto de fluxograma interativo para visualização de processos entre Despachante, Logística e TI.
---
## 📁 Estrutura
- index.html
- css/style.css
- js/script.js
- assets/
---
## 🚀 Funcionalidades
- Fluxograma interativo
- Painel de informações dinâmico
- Exportação em PNG
- Estrutura modular
---
##  Como usar
Abra o `index.html` no navegador.
---
##  Exportação
Clique em **Baixar Fluxo** para gerar a imagem.
---
##  Objetivo
Organizar e visualizar o fluxo de atendimento interno, facilitando o entendimento dos usuários sobre como proceder em cada etapa do processo.
